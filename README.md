# Website of kurone-kito

## Development

1. Install dependencies

   ```sh
   pnpm install
   ```

2. Start the development server

   ```sh
   pnpm run start
   ```

3. Build the project

   ```sh
   pnpm run build
   ```

## Environment Variables

`packages/fetcher` uses these values. Create a `.env` file or set them in
your environment.

- `CLIENT_ID`
- `CLIENT_SECRET`
- `ID_OTHERS`
- `ID_RELEASE`
- `ID_STREAMING`
- `REFRESH_TOKEN`

## System Requirements

- Node.js: Any of the following versions
  - LTS Jod (`^22.x.x`)
  - Latest (`>=24.x.x`)
